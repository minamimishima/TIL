#### セクション5: 44.開発でよく利用するイベントタイプ まで
- イベントリスナー
```
const Example = () => {
  const clickHandler = () => {
    alert('ボタンがクリックされました。');
  }

  return (
    <>
      <button onClick={clickHandler}>クリックしてね</button>
    </>
  );
};

export default Example;

```
onClickに渡すのは`clickHandler`で`()`はつけない！！！  
`()`をつけると関数を実行したことになってしまいクリックしなくてもalertが出る
