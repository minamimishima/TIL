## いろいろ
- `to_a`  
配列に変換  
<br>

- `join(sep)`  
配列の要素を文字列sepを間に挟んで連結  
<br>

- `for`  
繰り返し
```
例
for i in [1, 2, 3]
  print i*2, "\n"
end
```

- `each_cons(n)`  
要素を重複ありでn要素ずつに区切り、ブロックに渡して繰り返し
```
例
(1..10).each_cons(3){|v| p v }
# => [1, 2, 3]
#    [2, 3, 4]
#    [3, 4, 5]
#    [4, 5, 6]
#    [5, 6, 7]
#    [6, 7, 8]
#    [7, 8, 9]
#    [8, 9, 10]
```

- `drop`  
配列の先頭のn要素を捨てて、残りの要素を配列として返す
```
例
a = [1, 2, 3, 4, 5, 0]
a.drop(3)             # => [4, 5, 0]

# 変数aの値は変化しない
a                     # => [1, 2, 3, 4, 5, 0]
```
