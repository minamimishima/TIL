#### Ruby スキルチェック（Cランク） 3問 

- ハッシュとシンボルについて
```
currencies = { 'japan' => 'yen', 'us' => 'dollar', 'india' => 'rupee' }
```
この場合日本の通貨を取得したければ`currencies['japan']`

```
currencies = { 'japan': 'yen', 'us': 'dollar', 'india': 'rupee' }
```
この場合日本の通貨を取得したければ`currencies[:japan]`  
これは下記の書き換えだから（ハッシュのキーがシンボルになっている）
```
currencies = { :japan => 'yen', :us => 'dollar', :india => 'rupee' }
```
